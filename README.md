# Simple React Quiz App

Welcome! 👋  
This repository contains a **React Quiz App** built for learning and practicing frontend development with React.js.

## 🔹 Features
- Interactive quiz interface
- Multiple-choice questions
- Score tracking
- Option to restart the quiz

## 🔹 Technologies Used
- React.js (Functional Components, useState)
- JavaScript (ES6)
- HTML5 & CSS3 for styling

🔹 How to Run

1.Clone the repository:

git clone https://github.com/thisismanikanta/quiz-app.git

2.Navigate into the project folder:

cd quiz-app

3.Install dependencies:

npm install

4.Start the development server:

npm start

5.Open the app in your browser:

The app will run at http://localhost:3000

You can now interact with your React Quiz App! 🎉
